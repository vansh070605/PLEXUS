Universal Media Gateway - SARVAM UMG

Scalable Universal Media Gateway for Seamless Multi-Network Connectivity

Overview

SARVAM Universal Media Gateway is a flexible application, that connects VoIP, GSM, and PSTN networks. It allows businesses to manage calls across these platforms easily. With features like advanced call routing and secure connections, it adapts to changing communication needs. Suitable for businesses of all sizes, SARVAM UMG ensures smooth and reliable voice communication in hybrid setups.

USPs

Ports, Trunks, and Subscribers

  • Universal Media Gateway with up to 240 FXS, 192 FXO, 48 GSM/3G/4G Ports, and
    250 SIP Trunks
  • 250 SIP Trunks
  • Versatile Port Configuration for Analog and SIP Trunks
  • Scalable Capacity to handle High Volume of Subscribers
  • Efficient Call Routing and Management for Subscribers and Trunks

Universal Connectivity

  • Universal Media Gateway that Supports 4G, ISDN, VoIP, PSTN/CO/FXO on a Single Platform
  • Support for Multiple Network Protocols
  • Integration with Diverse Communication Systems and Networks
  • Future-ready Flexible Platform

Universal Design Architecture

  • Complete Flexible Solution
  • Universal Media Gateway Works even in Standalone Mode
  • Future-proof Design for Hardware Upgrades and Replacements

Features

Detailed Records of 2000 Calls

Universal Media Gateway provides comprehensive insights into call history, duration, and other call-related data to effortlessly track and analyze communication patterns

Call Hold

Put an on-going conversation on hold and call another person, or receive a call from another person. Retrieve the earlier call after ending the ongoing call.

Call Toggle

Universal Media Gateway allows you to have two simultaneous telephone conversations, talking to two people alternately.

SNMP Monitoring

Allows centralized monitoring and tracking of device performance, status, and network health for effective troubleshooting and optimization

Supports SRTP for Secure Transmission Over SIP

Ensure secure transmission of voice and data over SIP protocol with SRTP (Secure Real-time Transport Protocol) through Universal Media Gateway

Network and BCCH Locking

Enhance security and prevent unauthorized access by implementing network and BTS locking in the Gateway

Penta-band GSM and 4G band Network Operation

Experience wide compatibility and reliable network operation with Penta-band GSM and 4G band support with Universal Media Gateway

Least Cost Routing

Optimize call costs and choose the most cost-effective routes with Least Cost Routing in the Gateway

Hotline Extension Dialing

Universal Media Gateway connects the FXS Port user immediately to a particular number, every time the user lifts up the receiver of the phone
